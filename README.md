# Portfolio Website

A simple personal portfolio website built using **HTML** and **CSS**.

## Features

- Responsive layout (header/nav, sections, and cards)
- Dark theme styling with a **theme switch** control
- Sections included:
  - Home (intro + profile image + social links)
  - About (summary + “Read more” button)
  - Skills (animated/visible progress bars)
  - Certificates (certificate cards + links)
  - Projects (project image cards)
  - Contact (contact form UI)
  - Footer (call/email CTAs + back-to-top button)

## Tech Stack

- **HTML5**
- **CSS3**
- Font Awesome (via CDN) for social icons

## Project Structure

- `Home.html` – Main webpage
- `style.css` – All styling
- `images/` – Images, project thumbnails, and PDFs used by the page

## How to Run

Since this is a static site:

1. Open `Home.html` in any browser.
2. Ensure the `images/` folder stays in the same directory as `Home.html`.

## Assets

The site uses images and PDFs from the `images/` directory, including:

- `bg.avif` (background)
- `logoimg.png` (logo)
- `profile image.jpg` (profile)
- `*.pdf` (resume and certificates)

## Customization

Edit the following files to customize the content:

- `Home.html`: your name, bio text, certificates, projects, and contact form placeholders.
- `style.css`: colors, spacing, typography, animations, and responsive breakpoints.
